## How to Download

1. Go to: https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE283090
2. Under Supplementary file, download: GSE283090_RAW.tar (1.8 GB)
3. Extract the tar file into this data/raw/ folder
4. Each sample will have three files:
   - barcodes.tsv.gz
   - features.tsv.gz
   - matrix.mtx.gz
